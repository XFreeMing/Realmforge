## 一、地图全貌

### 1. 世界结构

山海经的世界由 **五层同心结构** 组成，从中心向外依次为：

```
中原（中山经）→ 四方山系（南/西/北/东山经）→ 海内（海内四经）→ 海外（海外四经）→ 大荒（大荒四经）
```

- **中心**：帝之下都昆仑之丘（西北），帝之平圃槐江之山
- **边界**：四海（东海、南海、西海、北海）环绕
- **极远**：大荒之地，日月出入之所

#### 2.2 核心山脉

- **昆仑之丘** [山-006]：帝之下都，方八百里，高万仞。上有木禾（长五寻，大五围），九井玉槛，九门开明兽守之。赤水、河水、洋水、黑水出焉。弱水之渊环之，外有炎火之山。（西次三经/海内西经/大荒西经）
- **不周山** [山-007]：西北海之外，山而不合，名曰不周负子。两黄兽守之。（大荒西经）
- **日月出入之山**：
  - 日出：大言山 [山-008]、合虚山 [山-009]、明星山 [山-010]、鞠陵于天山 [山-011]、猗天苏门山 [山-012]、壑明俊疾山 [山-013]（大荒东经）
  - 日落：丰沮玉门山 [山-014]、龙山 [山-015]、日月山 [山-016]、鏖鏊钜山 [山-017]、常阳山 [山-018]、大荒之山 [山-019]（大荒西经）
  - 日月同出入：柜格之松 [山-020]（大荒西经）
- **钟山** [山-021]：烛龙（烛阴）所居，视为昼，瞑为夜，吹为冬，呼为夏，身长千里。（海外北经/大荒北经）
- **泰山/华山/衡山** [山-022]：太华之山削成四方，高五千仞，广十里（西一经）；衡山在南海之内（海内经）

#### 2.3 河流水系

**四大源流（皆出昆仑）**：
- **河水（黄河）** [水-001]：出昆仑东北隅 → 行其北 → 入渤海 → 出海外 → 入禹所导积石山（海内西经）
- **赤水** [水-002]：出昆仑东南隅 → 行其东北 → 西南流注南海厌火东（海内西经）
- **洋水/黑水** [水-003]：出昆仑西北隅 → 东行 → 又东北 → 南入海（海内西经）
- **弱水/青水** [水-004]：出昆仑西南隅 → 东 → 北 → 西南（海内西经）

**主要河流（海内东经详载）**：
- 岷三江 [水-005]：大江出汶山，北江出曼山，南江出高山
- 浙江 [水-006] 出三天子都，入海
- 淮水 [水-007] 出馀山，入海
- 湘水 [水-008] 出舜葬东南陬，入洞庭
- 汉水 [水-009] 出鲋鱼之山
- 渭水 [水-010] 出鸟鼠同穴山，东注河
- 洛水 [水-011] 出洛西山，东北注河
- 汾水 [水-012] 出上窳北，西南注河
- 济水 [水-013] 出共山南东丘，注渤海
- 泾水 [水-014] 出长城北山，北入渭
- 白水 [水-015] 出蜀，东南注江
- 沅水 [水-016] 出象郡镡城西，合洞庭
- 赣水 [水-017] 出聂都东山，东北注江
- 泗水 [水-018] 出吴东北，东南注东海

**特殊水域**：
- **汤谷/甘渊** [水-019]：十日所浴之处，上有扶桑树，九日居下枝，一日居上枝（海外东经/大荒东经）
- **大泽** [水-020]：方千里，群鸟所解（大荒北经）；方百里，群鸟所生及所解，在雁门北（海内西经）
- **从极之渊** [水-021]：深三百仞，冰夷恒都焉（海内北经）
- **昆仑南渊** [水-022]：深三百仞（海内西经）
- **流沙** [水-023]：出钟山，西行又南行昆仑之墟，西南入海（海内西经）

#### 2.5 重要地理区域

- **都广之野** [域-001]：西南黑水之间，后稷葬焉。百谷自生，冬夏播琴。鸾鸟自歌，凤鸟自舞。（海内经）
- **沃之野** [域-002]：凤鸟之卵是食，甘露是饮，百兽相群而处。（大荒西经）
- **大乐之野** [域-003]：夏后启于此舞九代，乘两龙，云盖三层。（海外西经）
- **诸夭之野** [域-004]：鸾鸟自歌，凤鸟自舞，凤凰卵民食之，甘露民饮之。（海外西经）
- **幽都之山** [域-005]：北海之内，黑水出焉。上有玄鸟、玄蛇、玄豹、玄虎、玄狐蓬尾。（海内经）
