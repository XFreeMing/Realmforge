# 河流水系

> 来源：[山海经_世界总览.md](../山海经_世界总览.md) 一.3

## 四大源流（皆出昆仑）

- **河水（黄河）** [水-001]：出昆仑东北隅 → 行其北 → 入渤海 → 出海外 → 入禹所导积石山
- **赤水** [水-002]：出昆仑东南隅 → 行其东北 → 西南流注南海厌火东
- **洋水/黑水** [水-003]：出昆仑西北隅 → 东行 → 又东北 → 南入海
- **弱水/青水** [水-004]：出昆仑西南隅 → 东 → 北 → 西南

## 特殊水域

- **汤谷/甘渊** [水-019]：十日所浴之处，上有扶桑树，九日居下枝，一日居上枝
- **大泽** [水-020]：方千里，群鸟所解
- **从极之渊** [水-021]：深三百仞，冰夷恒都焉
- **昆仑南渊** [水-022]：深三百仞
- **流沙** [水-023]：出钟山，西行又南行昆仑之墟，西南入海
