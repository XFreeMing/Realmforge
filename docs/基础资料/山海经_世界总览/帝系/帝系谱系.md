# 帝系谱系

> 来源：[山海经_世界总览.md](../山海经_世界总览.md) 二.6.4

## 帝俊系 [帝-001]

```
帝俊 ─┬─ 中容 ── 中容国
      ├─ 帝鸿 ── 白民
      ├─ 黑齿 ── 黑齿国
      ├─ 晏龙 ─┬─ 司幽 ── 司幽国
      │        └─ 琴瑟（发明）
      ├─ 三身 ── 义均 ── 巧倕（百巧）
      ├─ 后稷 ── 降百谷 ── 台蠒 ── 叔均（牛耕）
      ├─ 禺号 ── 淫梁 ── 番禺 ── 奚仲 ── 吉光（车）
      ├─ 季厘 ── 季厘国
      └─ 十二子（歌儛）
```

## 黄帝系 [帝-002]

```
黄帝 ─┬─ 骆明 ── 白马（鲧）
      ├─ 禺𧴆 ── 禺京（禺强，海神）
      ├─ 苗龙 ── 融吾 ── 弄明 ── 白犬（犬戎）
      └─ 始均 ── 北狄
```

## 炎帝系 [帝-003]

```
炎帝 ─┬─ 听訞 ── 炎居 ── 节并 ── 戏器 ── 祝融 ── 共工
       │                                        └─ 术器
       │                                        └─ 后土 ── 噎鸣 ── 十二岁
       └─ 伯陵 ─┬─ 鼓 ── 钟（发明）
                ├─ 延 ── 钟（发明）
                └─ 殳 ── 侯（发明）
```

颛顼 ─┬─ 老童 ─┬─ 重 ── 上天
