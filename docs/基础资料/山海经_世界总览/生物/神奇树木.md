# 神奇树木

> 来源：[山海经_世界总览.md](../山海经_世界总览.md) 二.4.1

| 名称 | 形貌/特征 | 出处 | 功效 |
|------|-----------|------|------|
| **建木** | 青叶紫茎，玄华黄实，百仞无枝，九欘九枸 | 海内经/海内南经 | 众帝所自上下，大皞爰过，黄帝所为 |
| **扶桑/扶木** | 柱三百里，叶如芥 | 大荒东经/海外东经 | 十日所浴，九日居下枝一日居上枝 |
| **若木** | 赤皮支干，青叶（或赤树青叶赤华） | 大荒北经/海内经 | 若水出焉 |
| **沙棠** | 状如棠，华黄赤实，味如李无核 | 西山经 | 可御水，食之使人不溺 |
| **白䓘** | 状如谷赤理，汗如漆，味如饴 | 南山经 | 食者不饥，可释劳，可血玉 |
| **迷谷** | 状如谷黑理，花四照 | 南山经 | 佩之不迷 |
| **甘木/甘华/甘柤** | 甘木：食之不死；甘华/甘柤：枝干皆赤 | 大荒南经 | 不死国食甘木 |
| **枫木** | 蚩尤所弃桎梏化为枫木 | 大荒南经 | — |
| **三株树** | 状如柏，叶皆为珠 | 海外南经 | 生赤水上 |
| **珠树/文玉树/玗琪树/不死树** | 昆仑开明北 | 海内西经 | 不死树：长生 |
| **木禾** | 长五寻，大五围 | 西山经/海内西经 | 昆仑之上 |
| **櫰木** | 状如棠，圆叶赤实，实大如木瓜 | 西山经 | 食之多力 |
