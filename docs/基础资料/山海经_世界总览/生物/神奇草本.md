# 神奇草本

> 来源：[山海经_世界总览.md](../山海经_世界总览.md) 二.4.2

|------|------|------|------|
| 薲草 | 状如葵，味如葱 | 西山经 | 食之已劳 |
| 杜衡 | 状如葵，臭如蘼芜 | 西山经 | 可以走马，食之已瘿 |
| 蓇蓉 | 叶如蕙，本如桔梗，黑华不实 | 西山经 | 食之使人无子 |

| 无条 | 状如槁茇，叶如葵赤背 | 西山经 | 可以毒鼠 |
| 薰华草 | — | 海外东经 | 朝生夕死 |
