# 大荒北经

## 概述

《大荒北经》是大荒四经的最后一部分，记载了北方大荒之地的山川、神怪、部族与传说。北方大荒极寒荒芜，是中国古代神话中最为荒远寒冷的区域。

## 内容结构

### 记载要素

- 山川地貌
- 神灵异兽
- 部族与国家
- 自然现象
- 神话传说
- 冰雪奇观

### 著名内容

- **东北海之外**：大荒之中有山
- **钟山**：有神人面蛇身而赤
- **北海之渚**：有神人面鸟身
- **幽都之山**：黑色世界
- **大泽**：千里之地
- **衡天**：有人名曰䲦
- **不咸山**：有肃慎氏之国

### 神话传说

- **蚩尤作兵伐黄帝**：黄帝乃令应龙攻之冀州之野
- **夸父逐日**：道渴而死，其身体所化为邓林
- **禹杀相柳**：其血腥不可以树五谷
- **黄帝女魃**：所居不雨
- **颛顼与共工之战**

### 奇异物产

- 极寒地带特有的生物
- 冰雪中生存的奇异动物
- 特殊的矿物与宝石
- 北方荒漠特有的植物
