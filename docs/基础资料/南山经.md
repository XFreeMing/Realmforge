# 南山经

## 概述

《南山经》是《山海经》五藏山经之首，记载了南方山系的地理、物产、神怪等内容。

## 内容结构

### 山系脉络

南山经自西向东分为三列山系：
- **南山一经（鹊山山系）**：自招摇之山至箕尾之山，共十山
- **南山二经（柜山山系）**：自柜山至漆吴之山，共十七山
- **南山三经（南禺之山山系）**：自天虞之山至南禺之山，共九山

### 记载要素

每座山通常记载以下内容：
- 山名及其相对位置
- 水系（发源之河流）
- 矿产（金、玉、铜、铁等）
- 植被（奇花异草）
- 动物（异兽、怪鸟、鱼蛇等）
- 神灵及其祭祀方式
- 祥瑞与灾异之兆

### 著名异兽

- **狌狌**：知人名，其状如禺而白耳
- **鹿蜀**：其状如马而白首，其文如虎而赤尾
- **类**：其状如狸而有髦，自为牝牡
- **猼訑**：其状如羊，马尾，其名自号
- **尚付**：其状如鸡而三首、六目、六足、三翼
