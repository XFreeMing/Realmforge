# 海内东经

## 概述

《海内东经》记载了海内东方的诸国、地理、物产与神怪。东方海内临海，多有与水系、河流、湖泊相关的详细记载。

## 内容结构

### 记载要素

- 国名及其方位
- 地理山川
- 河流水系（记载最为详细）
- 物产资源
- 神怪形象
- 传说故事

### 著名内容

- **钜燕**：燕国所在
- **貊国**：东北方之国
- **朝鲜**：东海之内
- **列阳**：东海之内
- **三天子鄣山**：在闽西海中
- **浙江**：出三天子鄣山
- **沅水**：出象郡镡城西
- **资水**：出零陵山
- **湘水**：出零陵山

### 水系记载

海内东经对水系的记载最为系统详细：
- 各大河流的发源地
- 河流流经的地区
- 河流的汇入之处
- 河流长度（以里计）
- 沿途的地理特征

### 奇异物产

- 东方沿海特有生物
- 水产资源
- 特殊的矿物与宝石
