# 海内南经

## 概述

《海内南经》记载了海内南方的诸国、地理、物产与神怪。与海外经描述远方异域不同，海内经记载的是相对靠近中原的南方地区。

## 内容结构

### 记载要素

- 国名及其方位
- 地理山川
- 河流水系
- 物产资源
- 神怪形象
- 传说故事

### 著名内容

- **氐人国**：人面而鱼身，无足
- **巴蛇**：吞象，三岁而出其骨
- **闽**：海中地区
- **瓯**：海中地区
- **林氏国**：出珍兽
- **兕**：其状如牛，苍黑一角
- **犀**：其状如牛而黑

### 河流水系

- 湘水：洞庭之山，帝之二女居之
- 赣水：相关地理
- 郁水：相关地理

### 奇异物产

- 南方特有的珍禽异兽
- 热带与亚热带植物
- 水产与矿物
