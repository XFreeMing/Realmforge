# 海外南经

## 概述

《海外南经》是《山海经》海外八经之首，记载了南方海外诸国、异人、奇物的地理与风物。与山经不同，海外经更侧重于描述远方异国的风俗民情和奇异之人。

## 内容结构

### 记载要素

- 国名及其方位
- 国民形貌特征
- 异国风俗习惯
- 特产与奇物
- 神怪与传说
- 地理环境描述

### 著名异国

- **结胸国**：其人胸骨突出
- **羽民国**：其人身生羽毛，能飞
- **讙头国**：其人面有翼，以捕鱼为生
- **厌火国**：其人口中能吐火
- **三苗国**：其人三首一身
- **贯胸国**：其人胸前有孔
- **交胫国**：其人足胫相交
- **不死民**：其人长寿不死
- **岐舌国**：其人舌头分叉
- **三首国**：其人一身三首

### 奇异物产

- 各种神奇的动植物
- 特殊的矿物与宝石
- 奇异的自然现象
