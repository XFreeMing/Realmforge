# 大荒东经

## 概述

《大荒东经》是大荒四经之一，记载了东方大荒之地的山川、神怪、部族与传说。大荒经的内容比海外经和海内经更加荒远，充满了浓厚的神话色彩。

## 内容结构

### 记载要素

- 山川地貌
- 神灵异兽
- 部族与国家
- 自然现象
- 神话传说
- 日月星辰运行

### 著名内容

- **大言山**：日月所出之山
- **合虚山**：日月所出
- **明星山**：日月所出
- **鞠陵于天山**：日月所出
- **猗天苏门山**：日月所出
- **壑明俊疾山**：日月所出
- **东海之渚**：有神人面鸟身，名曰禺䝞
- **应龙**：处南极，杀蚩尤与夸父

### 神话传说

- **黄帝与蚩尤之战**：应龙蓄水，蚩尤请风伯雨师纵大风雨，黄帝乃下天女曰魃
- **夔**：东海流波山有神，状如牛，苍身而无角，其声如雷
- **雷神**：龙身而人头，鼓其腹
- **禺京**：人面鸟身，珥两青蛇

### 奇异物产

- 东方大荒特有的神怪
- 与日月出入相关的神山
- 特殊的自然现象
