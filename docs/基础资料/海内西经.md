# 海内西经

## 概述

《海内西经》记载了海内西方的诸国、地理、物产与神怪。西方海内多昆仑神话相关内容，是中国古代神话的核心区域之一。

## 内容结构

### 记载要素

- 国名及其方位
- 地理山川
- 河流水系
- 物产资源
- 神怪形象
- 传说故事

### 著名内容

- **昆仑之虚**：帝之下都，方八百里，高万仞
- **开明兽**：身大类虎而九首，皆人面，东向立昆仑上
- **河水**：出昆仑东北隅
- **赤水**：出昆仑东南隅
- **洋水**：出昆仑西北隅
- **黑水**：出昆仑西北隅
- **弱水**：环绕昆仑之渊
- **醴泉**：昆仑之虚上的甘泉
- **凤凰**：昆仑山上的神鸟
- **鸾鸟**：自歌自舞的神鸟

### 神话传说

- **昆仑天梯**：连接天地的通道
- **建木**：众帝所自上下之树
- **后稷**：农耕之神的传说
- **窫窳**：被贰负之臣所杀的神

### 奇异物产

- 昆仑山上的神奇草木
- 西方特有的玉石
- 神兽与灵禽
