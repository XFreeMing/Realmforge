# 北山经

## 概述

《北山经》是五藏山经的第三部分，记载了北方山系的地理、物产、神怪等内容。北方多寒冷，记载中有较多与冰雪、严寒相关的内容。

## 内容结构

### 山系脉络

北山经分为三列山系：
- **北山一经（单狐山山系）**：自单狐之山至隄山，共二十五山
- **北次二经（管涔山山系）**：自管涔之山至敦题之山，共十七山
- **北次三经（太行山山系）**：自太行之山至无逢之山，共四十六山

### 记载要素

- 山名及地理位置
- 河流及其流向
- 矿产与玉石
- 耐寒植物与奇草
- 北方特有的动物
- 神灵与祭祀
- 北方特有的自然现象描述

### 著名异兽

- **诸犍**：其状如豹而长尾，人首而牛耳
- **肥遗**：其状如蝮蛇而四翼
- **鵸䳜**：其状如乌，五采而赤文
- **长蛇**：其状如巴蛇，其毛如彘鬣
- **窫窳**：其状如牛而赤身、人面、马足
