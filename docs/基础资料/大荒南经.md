# 大荒南经

## 概述

《大荒南经》是大荒四经之一，记载了南方大荒之地的山川、神怪、部族与传说。南方大荒炎热潮湿，记载中有许多与酷暑、瘴气、虫蛇相关的内容。

## 内容结构

### 记载要素

- 山川地貌
- 神灵异兽
- 部族与国家
- 自然现象
- 神话传说
- 奇异草木

### 著名内容

- **柜涂山**：南海之中
- **羽山**：鲧所殛之处
- **驩头国**：其民人面有翼
- **赤水**：南方重要河流
- **不庭山**：有人曰季格，名曰季格
- **载民之国**：有人曰王亥，两手操鸟
- **因民国**：有人姓曰摇
- **季禺国**：颛顼之子

### 神话传说

- **鲧禹治水**：鲧窃帝之息壤以堙洪水
- **羲和浴日**：羲和者，帝俊之妻，生十日，浴于甘渊
- **三身国**：帝俊妻娥皇生此三身之国

### 奇异物产

- 南方炎热地带特有的生物
- 瘴气与毒物
- 神奇的草木
