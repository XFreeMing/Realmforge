# 大荒西经

## 概述

《大荒西经》是大荒四经之一，记载了西方大荒之地的山川、神怪、部族与传说。西方大荒包含大量昆仑神话体系的内容，是中国古代神话的核心区域。

## 内容结构

### 记载要素

- 山川地貌
- 神灵异兽
- 部族与国家
- 自然现象
- 神话传说
- 日月星辰运行

### 著名内容

- **大荒之山**：日月所入之山
- **龙山**：日月所入
- **日月山**：天枢也
- **鏊鏊巨山**：日月所入
- **常阳山**：日月所入
- **大荒之山**：有人反臂，名曰女丑
- **西海之渚**：有神人面鸟身
- **昆仑之丘**：其外有炎火之山

### 神话传说

- **女娲之肠**：有神十人，名曰女娲之肠
- **夏后开（启）**：上三嫔于天，得九辩与九歌以下
- **颛顼生老童**：老童生重及黎
- **禹攻共工国**
- **成汤伐夏桀**
- **西王母**：穴处，戴胜，有豹尾虎齿

### 奇异物产

- 西方大荒特有的神怪
- 与日月入处相关的神山
- 昆仑体系的神奇草木与玉石
