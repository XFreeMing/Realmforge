# 海内北经

## 概述

《海内北经》记载了海内北方的诸国、地理、物产与神怪。北方海内多寒冷之地，与戎狄等北方民族相关的记载较多。

## 内容结构

### 记载要素

- 国名及其方位
- 地理山川
- 河流水系
- 物产资源
- 神怪形象
- 传说故事

### 著名内容

- **犬戎国**：戎族，人面兽身
- **鬼国**：其民为鬼
- **穷奇**：其状如虎有翼
- **蜪犬**：其状如犬
- **林氏国**：出珍兽
- **大蜂**：其大如壶
- **朱獳**：其状如鱼而有翼
- **驺吾**：林氏国出珍兽，五采毕具，尾长于身

### 神话传说

- **黄帝与蚩尤之战**相关地理
- **禹杀相柳**：禹杀相柳于昆仑之北
- **贰负之臣**：杀害窫窳的故事

### 奇异物产

- 北方寒冷地带特有生物
- 草原与荒漠动植物
- 特殊的矿物与宝石
