# 西山经

## 概述

《西山经》是五藏山经的第二部分，记载了西方山系的地理、物产、神怪等内容，篇幅宏大，内容丰富。

## 内容结构

### 山系脉络

西山经分为四列山系：
- **西山一经（华山山系）**：自钱来之山至騩山，共十九山
- **西次二经（钤山山系）**：自钤山至莱山，共十七山
- **西次三经（崇吾山山系）**：自崇吾之山至翼望之山，共二十三山
- **西次四经（阴山山系）**：自阴山至弇山，共十九山

### 记载要素

- 山名及地理方位
- 河流源头及流向
- 矿产资源（玉、金、铜、铁、雄黄等）
- 奇异草木
- 珍禽异兽
- 神灵形象与祭祀仪式
- 历史传说与人物故事

### 著名异兽

- **肥遗**：其状如鹑，黄身而赤喙，食之已疠
- **𪄀如**：其状如枭而人面
- **数斯**：其状如鸱而人足
- **狡**：其状如犬而豹文，其音如吠犬
- **胜遇**：其状如鹿而白尾，其音如录

### 重要神灵

- **白少昊**：西方之神
- **蓐收**：金神，人面虎爪，执钺
