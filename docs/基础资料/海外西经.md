# 海外西经

## 概述

《海外西经》记载了西方海外的诸国、异人、奇物的地理与风物。西方海外多神话色彩浓厚的记载，包含许多著名的神话传说。

## 内容结构

### 记载要素

- 国名及其方位
- 国民形貌特征
- 异国风俗习惯
- 特产与奇物
- 神怪与传说
- 地理环境描述

### 著名异国

- **结匈国**：其人胸部隆起
- **巫咸国**：以巫咸为首，群巫往来于此
- **女子国**：全为女子，浴于玄池而孕
- **轩辕国**：人面蛇身，尾交首上
- **白民国**：其人披发 white，戴玉
- **长臂国**：其人手臂极长
- **一臂国**：其人只有一臂
- **奇肱国**：其人只有一臂三目
- **三身国**：其人一身三首
- **丈夫国**：全为男子

### 著名神话

- **刑天**：与天帝争神，被断首，以乳为目，以脐为口，操干戚而舞
- **夏后启**：乘两龙，左手操环，右手操璜
- **大乐之野**：夏后启于此舞九代

### 奇异物产

- 各种神奇的动植物
- 西方特有的矿物宝石
- 奇特的自然现象
