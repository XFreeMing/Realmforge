# 海外北经

## 概述

《海外北经》记载了北方海外的诸国、异人、奇物的地理与风物。北方海外多寒冷之地，记载中有许多与冰雪、极夜相关的内容。

## 内容结构

### 记载要素

- 国名及其方位
- 国民形貌特征
- 异国风俗习惯
- 特产与奇物
- 神怪与传说
- 地理环境描述

### 著名异国

- **无䏿之国**：其人无肠
- **柔利国**：其人一手一足，反膝
- **深目国**：其人深目，一手一足
- **无肠国**：其人腹中无肠
- **聂耳国**：其人两手聂耳
- **跂踵国**：其人行走时足跟不落地
- **拘瘿国**：其人颈有瘿瘤
- **一目国**：其人面只有一目
- **钟山之神**：人面蛇身，赤色

### 著名神话

- **烛龙（烛阴）**：钟山之神，人面蛇身而赤，视为昼，瞑为夜，吹为冬，呼为夏
- **相柳**：共工之臣，九首蛇身，自环其山
- **夸父**：与日逐走，道渴而死，其杖化为邓林

### 奇异物产

- 北方寒冷地带特有生物
- 冰雪中生存的奇异动物
- 特殊的矿物与宝石
