# 海外东经

## 概述

《海外东经》记载了东方海外的诸国、异人、奇物的地理与风物。东方海外临海，多有与水、日出相关的记载。

## 内容结构

### 记载要素

- 国名及其方位
- 国民形貌特征
- 异国风俗习惯
- 特产与奇物
- 神怪与传说
- 地理环境描述

### 著名异国

- **䢀国**：其人黑齿，食稻啖蛇
- **玄股国**：其人大腿为黑色
- **雨师妾国**：其人黑，两手各操一蛇
- **毛民国**：其人身生毛发
- **劳民国**：其人面貌憔悴
- **大人国**：其人形体巨大
- **君子国**：其人好让不争
- **黑齿国**：其人牙齿漆黑
- **汤谷**：日出之处，上有扶桑树

### 著名神话

- **扶桑树**：生于汤谷，十日所浴，九日居下枝，一日居上枝
- **禹虢**：禹所生之地
- **虹**：在虹北，各有两首

### 奇异物产

- 东方沿海特有生物
- 与水和日出相关的奇物
- 特殊的矿物与宝石
