# 中山经

## 概述

《中山经》是五藏山经的最后一部分，也是篇幅最大的部分，记载了中原地区山系的地理、物产、神怪等内容。因其地处中原，记载尤为详实。

## 内容结构

### 山系脉络

中山经分为十二列山系，涵盖地域最广：
- **中山一经**：自䳀山至丙山，共九山
- **中山二经**：自辉诸之山至葌山，共九山
- **中山三经**：自敖岸之山至明山，共九山
- **中山四经**：自釐山至几山，共十二山
- **中山五经**：自苟床之山至蔓渠之山，共九山
- **中山六经**：自景山至荆山，共十二山
- **中山七经**：自苦山至大騩之山，共十六山
- **中山八经**：自景山至琴鼓之山，共十六山
- **中山九经**：自岷山至风雨之山，共十二山
- **中山十经**：首山至丙山等
- **中山十一经**：涵盖更广区域
- **中山十二经**：中山经末卷

### 记载要素

- 山名及地理方位关系
- 河流源流及汇入
- 矿产资源
- 草木物产
- 鸟兽虫鱼
- 神灵形貌
- 祭祀礼仪

### 著名异兽

- **马腹**：其状如人面而龙身
- **鸱**：其状如鸮而人面
- **犀渠**：其状如牛，苍身，其音如婴儿
- **𤝱**：其状如獳犬而有鳞
- **夫诸**：其状如鹿而白尾
