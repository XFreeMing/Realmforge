# 东山经

## 概述

《东山经》是五藏山经的第四部分，记载了东方山系的地理、物产、神怪等内容。东方临海，多有与水相关的记载。

## 内容结构

### 山系脉络

东山经分为四列山系：
- **东山一经（樕鼄山山系）**：自樕鼄之山至竹山，共十二山
- **东次二经（空桑山山系）**：自空桑之山至邽山，共十七山
- **东次三经（尸胡山山系）**：自尸胡之山至无皋之山，共九山
- **东次四经（北号山山系）**：自北号之山至跂踵之山，共八山

### 记载要素

- 山名及其地理关系
- 河流及入海之处
- 矿产（玉、铜、铁等）
- 东方特有的草木
- 水产与陆生异兽
- 神灵及其形貌
- 祭祀方法与供品

### 著名异兽

- **蜚**：其状如牛而白首，一目而蛇尾
- **精精**：其状如马而牛尾
- **絜钩**：其状如凫而鼠尾
- **獙獙**：其状如狐而有翼
- **薄鱼**：其状如鳣鱼而一目
