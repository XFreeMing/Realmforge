# 海内经

## 概述

《海内经》是《山海经》的最后一篇，记载了海内的山川、地理、物产、部族、神怪及神话传说。作为全书的收尾，海内经既有对海内地理的总结性描述，也包含了大量重要的神话传说。

## 内容结构

### 记载要素

- 山川河流
- 地理疆域
- 部族分布
- 物产资源
- 神怪形象
- 神话传说
- 器物发明

### 著名内容

- **东海之内**：有流波之山
- **南海之内**：有菌山、桂山
- **西南黑水之间**：有都广之野
- **都广之野**：后稷葬焉，爰有膏菽、膏稻、膏黍
- **建木**：在都广，众帝所自上下
- **赤水之东**：有苍梧之野
- **九嶷山**：舜所葬处
- **封渊**：有神蛇

### 神话传说

- **黄帝生骆明**：骆明生白马，白马是为鲧
- **鲧复生禹**：帝乃命禹卒布土以定九州
- **帝俊赐羿彤弓素矰**：以扶下国
- **晏龙生司幽**：思士不妻，思女不夫
- **鼓延始为钟**：为乐风
- **番禺生舟**：舟船之发明
- **吉光始为车**：车辆之发明

### 器物发明

- 鼓：晏龙所制
- 钟：鼓延始为
- 舟：番禺所生
- 车：吉光所作
- 弓矢：般所制
- 耒耜：后稷所播

### 奇异物产

- 海内各地的珍禽异兽
- 神奇的草木
- 特殊的矿物与宝石
