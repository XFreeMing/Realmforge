## Section 1：游戏核心循环框架

### 完整 Session 流程（15-20 分钟）

```
打开 app → 检查世界状态 → 拍照获得材料 → 锻造武器 → 探索战斗 → 修复世界 → 保存
 [1min]       [1min]          [3min]          [5min]       [5min]      [2min]     [1min]
```

### 各系统串联关系

```
现实世界(拍照) → 材料 → 锻造工坊(设计+锻造) → 武器装备 → 山海世界探索
     ↑                                                    │
     │              新材料 + 知识/图鉴                     │
     └──────────── 妖兽掉落 + 世界修复 ← 探索/战斗/安抚 ←──┘
```

### 动机设计

| 阶段 | 孩子为什么想做 | 反馈 |
|------|-------------|------|
| 拍照 | "看看今天能找到什么新材料" | AI 识别 + 神话转化动画 + 材料卡片 |
| 锻造 | "用新材料做更厉害的武器" | 锻造小游戏 + 武器诞生光芒 + 属性报告 |
| 探索 | "新区域有什么妖兽和秘密" | 遭遇妖兽 + 神话故事 + 战斗成就感 |
| 战斗 | "试试新武器厉害不厉害" | 伤害数字 + 五行相克教学 + 材料掉落 |
| 世界修复 | "把灰色区域变回彩色" | 视觉变化（灰→彩）+ NPC 感谢 + 解锁内容 |

## Section 2：两大阵营玩法细化

### 统一命名

使用 **焱族（Yan/Flame）** 和 **渊族（Yuan/Abyss）**。技术文档 `faction` 字段改为 `yan / yuan`。

### 核心差异矩阵

| 维度 | 焱族 (Yan) | 渊族 (Yuan) |
|------|-----------|------------|
| 锻造哲学 | 快速、大胆、实验性 | 慢速、精准、和谐性 |
| 锻造偏好 | 武器/攻击型装备 | 护甲/辅助型装备 |
| 五行倾向 | 火、金 | 水、木 |
| 妖兽策略 | 战斗击杀为主 | 安抚驯服为主 |
| 专属技术 | 熔变（材料融合新属性） | 渊铸（额外属性微调槽） |
| 攻击加成 | 武器攻击力 +10% | 护甲防御力 +10%，耐久 +10% |
| 暴击 | 暴击率 +5% | 暴击率 -5% |
| 安抚难度 | 妖兽安抚 +10% 难度 | 妖兽安抚 -10% 难度 |
| 相克惩罚 | 相克材料组合惩罚减半 | 正常 |
| 相生加成 | 正常 | 五行相生组合加成 +15% |

### 阵营不可变规则

- 选择后不可切换（保持身份认同）
- 每周可访问对方领地学习技术（效果降低 30%）
- 大炉赛（每周一次）：获胜阵营全员获得"炉火印记"（24h 锻造加成 5%）

## Section 3：世界动态扩展机制

### 五境 ↔ Grid 映射

```
五境 = 大区（Realm Zone），每个大区包含多个 Grid Chunk

山境：熔炉谷(3) + 不周山(2) + 石语洞(2) = 7 chunks
海境：珊瑚城(3) + 归墟(2) + 龙宫遗迹(2) = 7 chunks
林境：建木(2) + 百草园(3) + 迷踪林(2) = 7 chunks
云境：风伯台(2) + 雷池(1) + 彩虹桥(1) = 4 chunks
梦境：万象殿(2) + 回音原(2) + 遗忘之角(2) = 6 chunks

总计: ~28 个 Grid Chunks
```

### 拍照决策逻辑

```
IF 物体是「动物/昆虫」→ 80% 妖兽遭遇, 15% 材料, 5% realm key
IF 物体是「植物/自然物」→ 60% 材料, 30% 世界种下灵种, 10% 妖兽遭遇
IF 物体是「人造物」→ 80% 材料, 15% 解锁配方, 5% 特殊事件
IF 物体是「天气/风景」→ 材料 + realm key + 临时天气效果
IF 物体是「儿童绘画」→ 绘画生物进入世界
```

### 黯褪（The Fading）机制

```
世界活力值 = Σ(各 Realm 活力值 0-100)

活力值来源：
  拍照贡献 +2~5/次 | 修复碎片 +10/次 | 击败/安抚妖兽 +1~3/次 | 完成任务 +5/次

活力值衰减：
  每 24h -1（自然衰减）| 48h 无活跃 -3/天

活力值效果：
  0-20: 灰暗，妖兽稀少，材料减半
  21-40: 部分色彩，基础妖兽
  41-60: 正常色彩，完整妖兽池
  61-80: 鲜艳，稀有妖兽
  81-100: 光辉，传说妖兽，材料 ×1.5

Floor: 20%（仅因不活跃不会低于此值，防止不可赢状态）
```

### Realm 旅行

- 通过 **川（World River）** 在各 Realm 间旅行
- 旅行方式：骑乘巨大妖兽的背
- 旅行是加载画面 + 简短动画（非可探索空间）
- Realm 解锁条件：故事进度 + 世界活力值 + 特定材料

## Section 4：新手引导系统

### 首次游玩流程（~18 分钟，可分两次 session）

```
Step 1: 家长设置 (2 min) → 账号 + 年龄 + 隐私 + PIN
Step 2: 碎镜觉醒 (30 sec) → 触碰镜子碎片
Step 3: 进入山海世界 (1 min) → 遇到第一个 NPC
Step 4: 第一次拍照 (3 min) → 获得第一份材料
Step 5: 第一次锻造 (5 min) → 获得第一把武器
Step 6: 第一次探索 (3 min) → 遭遇妖兽 → 五行相克教学
Step 7: 第一次世界修复 (2 min) → 灰色变彩色
Step 8: 阵营选择 (1 min) → 焱族 or 渊族
```

### 渐进式功能解锁

| Session | 解锁 |
|---------|------|
| 第 1 次 | 拍照→材料→锻造→战斗→世界修复（核心循环） |
| 第 2 次 | 自由绘画设计 |
| 第 3 次 | 妖兽图鉴 + 五行详解 |
| 第 4-5 次 | 妖兽安抚 + 伙伴系统 |
| 第 6-8 次 | 材料组合进阶 |
| 第 8-10 次 | 妖兽繁育 |
| 第 10+ 次 | 大炉赛（阵营竞赛） |
